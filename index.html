Develop Bosh Release inside Vagrant

Developing your application code, your stack configuration code and templates, and your base package descriptions within BOSH can be painful if you're doing it against real VMs. In this video, it shows an example of compiling/deploying a BOSH release within a base lucid64 Vagrant VM, without requiring any of BOSH to be running.

Creating bosh releases

Bosh is a great way to describe, deploy and manage complex systems. In this tutorial, we create a new bosh release (description of the complex system) and deploy it for a simple system - a single redis VM.

The tutorial uses bosh-gen and bosh-solo (on top of vagrant) to help rapidly develop the new bosh release.
